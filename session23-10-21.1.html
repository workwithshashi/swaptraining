JavaScript Datatype

var a = "Shashi"

JS has 2 type of Datatype

  • Primitive
    • Number //-> var a =10;
    • String //-> var a ="Shashi";
    • Booleas //-> var a =true/false;
    • Null //-> var a =null;
    • Undefined //-> var a =undefined;
  • Non Primitive
    • Function //-> var a = function abc(){};
    • Arrays //-> var a =[a,1,b];
    • Objects //-> var a =Object a;
